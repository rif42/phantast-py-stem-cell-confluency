# Data Model

## Entities

### Image
Represents the raw microscope image file loaded into the application.

### Pipeline
A saved sequence of processing operations (like a recipe) that can be applied to images.

### Step
A single operation within a pipeline (e.g., "CLAHE", "Grayscale", "PHANTAST") with specific parameters.

### Result
The outcome of running a pipeline on an image, including the processed image, mask, and confluency score.

### BatchJob
A collection of images being processed together using a single pipeline.

## Relationships

- Pipeline has many Steps
- Step belongs to one Pipeline
- BatchJob uses one Pipeline
- BatchJob processes many Images
- Result belongs to an Image and is generated by a Pipeline
