# Phantast Lab

This repository contains the front-end user interface for Phantast Lab, integrated as a desktop application using PyQt6.

## Getting Started

1. Install Python 3.9+
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python src/main.py
   ```

## Structure
- `src/main.py`: The entry point for the application.
- `src/shell/`: Contains the Application Shell (Main Window, Navigation, Header, Footer).
- `src/sections/`: Contains the modular feature screens (Image Navigation, Pipeline Construction, Batch Execution).
- `docs/`: Product overview and implementation guidance.
